## Guidenline

### Requirements

* Python installed: ``` Python version > 3.10 ```
* PyGame installed: ``` PyGame version > 1.9.5 ```

## Run the game

`python <file_path>/main.py`

## Create release

`pyinstaller.exe .\main.spec`
